Советник Expert предназначен для открытия ордеров с определенным шагом.
Советник не открывает отложенные ордера.
Для кажтого типа ордера (покупка/продажа) можно определить разные: шаг, объем, TakeProfir.
Также можно отключить открытие ордеров в зависимости от типа.
Для всех ордеров которые открывает советик указывается одно значение StopLoss. 
Если значение StopLoss или TakeProfir равно 0, то эти значения устанавливаться не будут.

Изменяемые данные, которые использует советник:
1) Buy: on/off - включение/отключение открытия ордеров на покупку.
2) Sell: on/off - включение/отключение открытия ордеров на продажу.
3) LotBuy: объем ордера на покупку.
4) LotSell: объем ордера на продажу.
5) stoploss: размер stoploss в пунктах.
6) TPBuy: размер takeprofit в пунктах для ордеров на покупку.
7) TPSell: размер takeprofit в пунктах для ордеров на продажу.
8) StepBuy: шаг открытия ордеров на покупку.
9) StepSell: шаг открытия ордеров на продажу.

Советник сохраняет свои значения при изменении периода графика, изменении шаблона графика, закрытии терминала.
